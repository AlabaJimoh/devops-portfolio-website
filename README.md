DevOps Portfolio Website
Overview

This project is a personal DevOps portfolio website built as part of my hands-on DevOps learning journey.

The purpose of this project is to demonstrate my ability to:

Work in a Linux environment using Ubuntu WSL2
Create and manage project files using VS Code
Use Git for version control
Document projects professionally
Build and test a simple website locally
Prepare projects for GitHub and future CI/CD integration

The website contains information about my skills, learning journey, projects, and future DevOps goal
Technologies Used
HTML

Used to create the structure and content of the website.

CSS

Used to style the website and improve its appearance.

Git

Used for version control, tracking changes, and managing project history.

Ubuntu WSL2

Used as the Linux development environment for building and testing the project.

VS Code

Used as the primary code editor for creating and managing project files.

Installation
Prerequisites
Ubuntu WSL2
Git
VS Code
Python 3


Clone Repository
git clone https://github.com/AlabaJimoh/devops-portfolio-website.git
Navigate to Project Directory
cd devops-portfolio-website
Start Local Web Server
python3 -m http.server 8000

Open in Browser
http://localhost:8000

The website should now be running locally.
